华中科技大学硬件综合训练项目
=======================
项目简介
--------

**项目名称：** MagicCube

- 我们在 FPGA 开发板上成功实现了一套支持重定向的多级中断流水线，支持 RISC-V 指令集。
- 同时，我们完成了 VGA 驱动与显示逻辑的开发，构建了清晰流畅的显示系统。
- 此外，我们还实现了交叉编译功能，提供了虚拟运行环境，以确保游戏开发与硬件实现能够高效并行推进。
- 最终，我们在 FPGA 开发板上“手搓”了一颗 CPU，并成功运行了小游戏 MagicCube，充分展示了硬件设计与软件开发的完美融合。
